# Comparatives

### Comparative adjectives

Add **-er**

| Adjective | Comparative |
|-----------|-------------|
| klein | kleiner |
| groot | groter |
| mooi | mooier |
| dik | dikker |
| ver | verder |

> Use **dan** for "than": *Hij is **groter dan** ik.*
>
> Adjectives ending in **-r** add **-d-** before -er: *duur → duurder*, *zwaar → zwaarder*.

### Superlative adjectives

Add **-st**

| Adjective | Superlative |
|-----------|-------------|
| klein | kleinst |
| groot | grootst |
| mooi | mooist |
| dik | dikst |

### Comparison of Adverbs

Adverbs use the same comparative/superlative endings as adjectives.

| Positive | Comparative | Superlative |
|----------|-------------|-------------|
| **snel** | sneller | het snelst |
| **vaak** | vaker | het vaakst |
| **graag** | liever | het liefst |
| **goed** | beter | het best |
| **veel** | meer | het meest |

In adverbial use, the superlative takes **het + -st**:

- *Hij rent **het snelst**.* — He runs the fastest.
- *Ik drink **het liefst** koffie.* — I most prefer to drink coffee.

## Common mistakes

- Using *als* for "than": ❌ *groter **als*** → ✅ *groter **dan*** (in standard Dutch).
