# Conversation flow

> Combinations like toch maar even or gewoon eens are normal in spoken Dutch. They soften commands and add nuance — learners often skip them and sound abrupt.

### Maintaining Conversation

| Dutch | English | Usage |
|-------|---------|-------|
| dus | so | En dus, wat wilde je zeggen? (So, what did you want to say?) |
| trouwens | by the way | Trouwens, hoe gaat het met je? (By the way, how are you?) |
| nog wat | also, another thing | En nog wat, bel me later. (Also, call me later.) |
| weet je | you know | Weet je, dat is interessant. (You know, that's interesting.) |
| kijk | look/see | Kijk, zo werkt het. (Look, this is how it works.) |
| luister | listen | Luister, ik moet je wat vertellen. (Listen, I need to tell you something.) |
| zeg | hey (calling) | Zeg, kun je me helpen? (Hey, can you help me?) |
| hé | hey | Hé, wat leuk! (Hey, how nice!) |
| pardon | pardon | Pardon, waar is het station? (Pardon, where is the station?) |
| excuseer | excuse me | Excuseer, kunt u dat herhalen? (Excuse me, can you repeat that?) |
| dus | so | Dus, dat was het. (So, that was it.) |
| eindelijk | finally | Eindelijk! Ik ben klaar. (Finally! I'm done.) |
| kortom | in short | Kortom, het was een disaster. (In short, it was a disaster.) |
| al met al | all in all | Al met al was het goed. (All in all, it was good.) |
| wat ik zeggen wil | what I want to say | Wat ik zeggen wil... (What I want to say...) |
| hoe dan ook | anyway | Hoe dan ook, ik kom morgen. (Anyway, I'm coming tomorrow.) |
| volgens mij | in my opinion | Volgens mij heb je gelijk. (In my opinion, you're right.) |
| ik vind dat | I think that | Ik vind dat we moeten gaan. (I think we should go.) |
| persoonlijk | personally | Persoonlijk vind ik het lekker. (Personally, I think it's nice.) |
| Ik denk dat … | I think that … | Ik denk dat het morgen regent. (I think that it will rain tomorrow.) |
| naar mijn idee | in my view | Naar mijn idee is het waar. (In my view, it's true.) |
| in mijn ogen | in my eyes/view | In mijn ogen heb je ongelijk. (In my view, you're wrong.) |
| wat mij betreft | as far as I'm concerned | Wat mij betreft hoeft dat niet. (As far as I'm concerned, that's not necessary.) |
| nou | well | Nou, ik weet het niet. (Well, I don't know.) |
| zeg maar | so to speak | Het is, zeg maar, een mix. (It's, so to speak, a mix.) |
| hoe zal ik het zeggen | how shall I put it | Hoe zal ik het zeggen... (How shall I put it...) |
| dus | so | En dus, ik was net... (So, I was just...) |
| wacht even | wait a moment | Wacht even, dat klopt niet. (Wait a moment, that's not right.) |
| effe | just (colloquial) | Effe kijken... (Just a moment...) |
| Ik hou van … | I love / I'm fond of … | Ik hou van koffie. (I love coffee.) |
| Ik vind … leuk / niet leuk(saai) | I (don't) like … | Ik vind voetbal leuk. (I like football.) |
| Ik doe graag … | I like doing … | Ik lees graag in de tuin. (I like reading in the garden.) |
| Heb je zin in...? | Do you feel like...? | Heb je zin in een kopje koffie? (Do you feel like a cup of coffee?) |

> *graag* is an adverb, not a verb — it slots between the conjugated verb and the rest: *Ik drink graag thee*.
>
> After Ik denk dat / Ik vind dat the verb moves to the end of the clause: *Ik denk dat het leuk is* — not *…dat het is leuk*.

### Agreement and Confirmation, Disagreement and Correction

| Dutch | English | Usage |
|-------|---------|-------|
| dat klopt | that's correct | Ja, dat klopt helemaal. (Yes, that's exactly right.) |
| zo is het | that's how it is | Tja, zo is het nu eenmaal. (Well, that's just how it is.) |
| dat mag | that's allowed | Natuurlijk, dat mag. (Of course, that's allowed.) |
| dat kan | that's possible | Ja hoor, dat kan. (Sure, that's possible.) |
| dat spreekt | that goes without saying | Dat spreekt vanzelf. (That goes without saying.) |
| inderdaad | indeed | Inderdaad, dat is waar. (Indeed, that's true.) |
| natuurlijk | of course | Natuurlijk, ik help je. (Of course, I'll help you.) |
| natuurlijk wel | of course yes | Natuurlijk wel! (Of course!) |
| uiteraard | naturally/certainly | Uiteraard ga ik mee. (Naturally, I'm coming along.) |
| ja hoor | yes indeed | Dat ja hoor. (Yes indeed.) |
| mee eens | I agree | Daar ben ik het mee eens. (I agree with that.) |
| precies | exactly | Precies! Zo bedoel ik. (Exactly! That's what I mean.) |
| eigenlijk | actually/in fact | Eigenlijk heb ik geen tijd. (Actually, I don't have time.) |
| niet echt | not really | Niet echt, nee. (Not really, no.) |
| echt niet | really not | Echt niet, nooit. (Really not, never.) |
| nee hoor | no (polite) | Dat nee hoor. (No, not at all.) |
| integendeel | on the contrary | Integendeel, het ging goed. (On the contrary, it went well.) |
| eerder | rather | Dat is eerder een voordeel. (That's rather an advantage.) |

### Polite negation and Hedging Phrases

| Dutch | English | Example |
|-------|---------|---------|
| niet echt | not really | Ik vind het **niet echt** lekker. (I don't really find it tasty.) |
| niet zo | not that… | Het is **niet zo** moeilijk. (It's not that difficult.) |
| niet bepaald | not exactly | Hij is **niet bepaald** vriendelijk. (He is not exactly friendly.) |
| liever niet | rather not | Ik wil **liever niet** gaan. (I'd rather not go.) |
| ik weet het niet zeker | I'm not sure | **Ik weet het niet zeker**, vraag het hem maar. (I'm not sure, just ask him.) |
| het is mogelijk | it's possible | **Het is mogelijk** dat ik later kom. (It's possible that I'll come later.) |
| dat zou kunnen | that could be | "Komt hij ook?" "**Dat zou kunnen**." ("Is he coming too?" "That could be.") |
| wie weet | who knows | **Wie weet** zien we elkaar nog eens. (Who knows, maybe we'll meet again.) |
| misschien wel | perhaps indeed | "Heb je honger?" "**Misschien wel**." ("Are you hungry?" "Perhaps indeed.") |
| ik denk van wel | I think so | "Komt het goed?" "**Ik denk van wel**." ("Will it be okay?" "I think so.") |
| ik hoop van wel | I hope so | "Lukt het je?" "**Ik hoop van wel**." ("Will you manage?" "I hope so.") |
| ik denk van niet | I don't think so | "Regent het morgen?" "**Ik denk van niet**." ("Will it rain tomorrow?" "I don't think so.") |

### Softener and Polite words

| Word | Effect | Example |
|------|--------|---------|
| alstublieft / alsjeblieft | please | *Mag ik een glas water, alstublieft?* |
| misschien | perhaps | *Heb je misschien een pen voor mij?* |
| even | for a moment | *Kun je me even helpen?* |
| toevallig | by any chance | *Heb je toevallig een paraplu?* |

The embedded clause goes verb-final.

| Wrapper | Example |
|---------|---------|
| Mag ik vragen … | *Mag ik vragen waar je vandaan komt?* |
| Zou je me kunnen vertellen … | *Zou je me kunnen vertellen hoeveel dat kost?* |
| Weet je toevallig … | *Weet je toevallig hoe laat de bus komt?* |
| Ik vroeg me af … | *Ik vroeg me af of er nog plekken zijn.* |

Soften the force of a statement.

| Dutch | English | Example |
|-------|---------|---------|
| een beetje | a little | *Het is een beetje moeilijk.* |
| nogal | quite / rather | *Het is nogal duur.* |
| best | quite / pretty | *Het is best goed.* |
| tamelijk | fairly | *Dat is tamelijk interessant.* |
| redelijk | reasonably | *Dat is redelijk duur.* |
| vrij | fairly / rather | *Dat is vrij goed.* |
